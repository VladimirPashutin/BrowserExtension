## Описание проекта

**Яндекс.Бизнес помощник** — расширение браузера, призванное помочь пользователям управлять отзывами о своей компании на Яндексе. Расширение предоставляет интерфейс боковой панели, который позволяет пользователям входить в систему, генерировать ответы на отзывы и применять эти ответы непосредственно на странице отзывов Яндекса. Расширение взаимодействует с API бэкэнда для выполнения различных операций.

## Базовые возможности

- **Login**: Регистрирует пользователя в API бекенда, без регистрации никакие другие функции недоступны, создать аккаунт можно на основной странице бекенда https://business.t3t.online.
- **Review Management**: Управление ответами на отзывы пользователей - сбор ответов со страницы компании, обращение к бекенду для формирования ответа на основе примнения функций искусственного интеллекта, прикрепление сгенерированных ответов к отзывам.
